# Synergy BO4 GSC Menu

A Black Ops 4 Zombies GSC Mod Menu that uses [Project BO4 / Shield](https://shield-bo4.gitbook.io/document)

This Menu is a Work-In-Progress, so expect changes and possibly some Bugs

If you use any Code from my Menu, your Menu also has to be Open-Source as per the [GNU GPLv3 License](https://github.com/SyndiShanX/Synergy-BO4-GSC-Menu/blob/main/LICENSE.md)

If you have any Issues/Suggestions, please Submit them in [Issues](https://github.com/SyndiShanX/Synergy-BO4-GSC-Menu/issues)

#### Preview:

![Mod Menu](https://syndishanx.github.io/Synergy-BO4-GSC-Menu/Synergy-BO4-GSC-Menu.png)

### Requirements: [Project BO4 / Shield](https://shield-bo4.gitbook.io/document)

#### Installation:

* You can install the Mod ingame on the Mod Manager Page

##### Manual Installation:

* Download the Latest ZIP from [Releases](https://github.com/SyndiShanX/Synergy-BO4-GSC-Menu/releases)

* Place the Synergy Folder in the ZIP into `project-bo4/mods`
